# google-keep-to-notion

## Usage

Create a `.dotenv` file with these properties:
```
NOTION_API_TOKEN=
NOTION_PAGE_ID=
GOOGLE_KEEP_TAKEOUT_DIR_PATH=
```

Run the tool:
```
npm install
npm start
```
